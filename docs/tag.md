# 标签页设置 (Tag)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 标签**

---

## 侧边栏设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示侧边栏** | 控制标签页是否显示侧边栏 | 关闭 |
| **定制侧边栏** | 启用后可单独配置，禁用则使用全局配置 | 关闭 |
| **侧边栏位置** | 左侧 / 右侧 | 右侧 |
| **小工具列表** | 拖拽排序添加小工具 | - |

### 可用小工具
欢迎卡片（天气）、作者卡片、博客统计、文章分类、标签云、热门文章、最新文章

---

## 文章列表风格

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **文章列表风格** | 卡片 / 列表 / 杂志 / 极简 | 极简 |
| **标签切换块风格** | 药丸 / 芯片 / 下划线 / 玻璃拟态 | 下划线 |

---

## 文章封面回退

控制标签列表的封面替代策略（默认不启用）。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用封面替代** | 无封面时使用指定图片 | 关闭 |
| **替代模式** | 无封面时替代 / 始终使用 | 无封面时替代 |
| **默认封面图片** | 上传替代封面图片 | - |

---

## 悬浮控制栏

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **定制悬浮控制栏** | 启用后可单独配置 | 关闭 |
| **Dock 菜单** | 选择菜单 | primary |
| **返回顶部按钮** | 是否显示 | 开启 |
