# 页脚设置 (Footer)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 页脚（全局）**

配置网站页脚的各项内容。

---

## 1. 站点信息设置

| 设置项 | 说明 |
|--------|------|
| **站点图标类型** | `隐藏` / `Iconify` / `SVG` / `上传图片` |
| **站点图标** | 页脚显示的站点图标 |
| **显示站点标题** | 是否显示页脚站点标题 |
| **自定义站点标题** | 留空使用站点标题 |
| **显示站点副标题** | 是否显示副标题描述 |
| **站点副标题** | 站点标题下方的描述文字 |
| **显示运行天数** | 是否显示网站运行天数统计 |
| **建站日期** | 网站建立日期（格式：YYYY-MM-DD） |

---

## 2. 底部导航设置

| 设置项 | 说明 |
|--------|------|
| **显示底部导航** | 是否显示页脚快速导航区域 |
| **导航标题** | 快速导航区域的标题 |
| **导航链接** | 配置快速导航的链接项（名称+链接） |

---

## 3. 社交链接设置

| 设置项 | 说明 |
|--------|------|
| **显示社交媒体** | 是否显示页脚社交媒体链接 |
| **社交媒体标题** | 社交媒体区域的标题（如"关注我们"） |
| **社交媒体链接** | 配置社交媒体链接（名称+链接+图标） |

### 内置社交媒体图标
- GitHub、Twitter/X、Telegram、邮箱、RSS 订阅

> **💡 提示**：图标通过 Iconify 图标库选择，支持任意 SVG 图标。

---

## 4. 版权信息设置

| 设置项 | 说明 |
|--------|------|
| **显示法律链接** | 是否显示隐私政策、使用条款等 |
| **法律链接** | 配置法律相关链接 |
| **版权文字** | 版权声明文字（如 "© 2025 Sky Blog"） |
| **显示 ICP 备案** | 是否显示 ICP 备案信息 |
| **ICP 备案号** | ICP 备案号码 |
| **显示公安备案** | 是否显示公安备案信息 |
| **公安备案号** | 公安备案号码 |
| **显示托管信息** | 是否显示 CDN 和托管服务信息 |
| **托管服务链接** | 配置 CDN 和托管服务链接 |

> **💡 说明**：中国大陆网站需要填写 ICP 备案和公安备案信息。
