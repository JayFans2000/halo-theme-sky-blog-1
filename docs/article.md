# 文章页设置 (Article)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 文章**

配置文章页的布局和功能。

---

## 1. 文章头部背景设置

| 设置项 | 说明 |
|--------|------|
| **背景图片来源** | `文章封面图` 或 `自定义图片` |
| **自定义背景图片** | 上传图片或输入 URL |
| **启用回退图片** | 当图片来源为空时自动使用回退图片 |
| **回退图片** | 默认背景图片（支持随机图片 API） |

---

## 2. 内容显示设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示文章摘要** | 在文章内容前显示摘要（打字机效果） | 开启 |

### 使用说明

- **开启**：在文章正文前显示摘要，带打字机动画效果
- **关闭**：隐藏摘要区域

### 注意事项

⚠️ **如果安装了 AI 摘要插件，建议关闭此选项**，避免显示两个摘要造成冗余。

---

## 3. 侧边栏设置

| 设置项 | 说明 |
|--------|------|
| **显示侧边栏** | 是否显示（包含目录、分类、标签等） |
| **定制侧边栏** | 启用后可单独配置，禁用则使用全局配置 |
| **侧边栏位置** | 左侧 / 右侧 |
| **小工具列表** | 拖拽排序 |

### 文章页可用小工具

| 小工具 | 说明 |
|--------|------|
| **文章目录** | 显示文章 TOC 目录（推荐） |
| **欢迎卡片（天气）** | 问候语和天气 |
| **作者卡片** | 博主信息 |
| **博客统计** | 统计数据 |
| **文章分类** | 分类列表 |
| **标签云** | 标签云展示 |
| **热门文章** | 热门文章列表 |
| **最新文章** | 最新文章列表 |

---

## 4. 文章底部交互

| 设置项 | 说明 |
|--------|------|
| **启用点赞** | 显示点赞按钮 |
| **启用分享** | 显示分享按钮（打开分享弹窗） |
| **启用的分享平台** | 选择分享平台，可拖拽排序 |
| **启用评论** | 显示评论区 |

### 支持的分享平台
微信、X (Twitter)、Telegram、Facebook、QQ、QQ 空间、微博、豆瓣、系统分享

---

## 5. 悬浮控制栏

| 设置项 | 说明 |
|--------|------|
| **启用定制 Dock** | 启用后可自定义，禁用则使用全局设置 |
| **Dock 样式** | 跟随全局 / 底部居中 / 右侧可折叠 |
| **文章页 Dock 菜单** | 选择菜单 |
| **启用目录抽屉** | 移动端从 Dock 打开文章目录 |
| **返回顶部按钮** | 是否显示 |

> **💡 提示**：目录抽屉在移动端或隐藏侧边栏时特别有用，让用户可以快速导航到文章任意章节。
