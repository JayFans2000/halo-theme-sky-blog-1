# 朋友圈设置 (Friends)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 朋友圈**

需安装 `plugin-friends` 插件。

> **📦 插件配置提示**：每页显示数量等参数请在「后台 → 插件 → 朋友圈 → 基本设置」中配置

---

## 页面设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **页面标题** | 朋友圈页面的标题 | "朋友圈" |
| **页面描述** | 朋友圈页面的描述 | "订阅朋友们的最新动态" |
| **列表样式** | 单列 / 双列 | 单列 |

---

## 侧边栏设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示侧边栏** | 启用后显示侧边栏，页面变为左右布局 | 关闭 |
| **定制侧边栏** | 启用后可单独配置此页面的侧边栏 | 关闭 |
| **侧边栏位置** | 左侧 / 右侧 | 右侧 |
| **小工具列表** | 拖拽排序添加小工具 | - |

### 可用小工具
欢迎卡片（天气）、作者卡片、博客统计、文章分类、标签云、热门文章、最新文章

---

## 悬浮控制栏

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **定制悬浮控制栏** | 启用后可单独配置 | 关闭 |
| **Dock 菜单** | 选择菜单 | primary |
| **返回顶部按钮** | 是否显示 | 开启 |
