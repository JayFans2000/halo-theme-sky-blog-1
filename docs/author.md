# 作者页设置 (Author)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 作者**

---

## 侧边栏设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示侧边栏** | 控制作者页是否显示侧边栏 | 开启 |
| **定制侧边栏** | 启用后可单独配置，禁用则使用全局配置 | 关闭 |
| **侧边栏位置** | 左侧 / 右侧 | 右侧 |
| **小工具列表** | 拖拽排序添加小工具 | - |

### 可用小工具
欢迎卡片（天气）、博客统计、文章分类、标签云、热门文章、最新文章

---

## 内容显示

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **文章每页数量** | 设置作者页文章列表每页显示的数量（5-50） | 10 |
| **显示瞬间** | 在作者页面显示该作者发布的瞬间 | 开启 |
| **瞬间显示数量** | 设置瞬间区块最多显示的条数（3-20） | 6 |
| **默认显示** | 文章 / 瞬间 | 文章 |

---

## 悬浮控制栏

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **定制悬浮控制栏** | 启用后可单独配置 | 关闭 |
| **Dock 样式** | 底部居中 / 右侧可折叠 | 右侧可折叠 |
| **Dock 菜单** | 选择菜单 | primary |
| **返回顶部按钮** | 是否显示 | 开启 |
