# 通用设置 (General)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 通用（全局）**

本页面包含影响整个站点的全局配置选项。

---

## 1. 页面加载屏幕

在页面加载时显示加载动画，避免样式闪烁和页面变形。

| 设置项 | 说明 |
|--------|------|
| **启用加载屏幕** | 开关控制是否显示加载动画 |
| **动画来源** | `内置 CSS 动画` 或 `自定义 GIF 动图` |
| **CSS 动画样式** | Spinner / Dots / Bars / Ring（四种内置样式） |
| **上传 GIF 动图** | 自定义动画，建议不超过 100KB |
| **加载文字** | 动画下方的提示文字，默认 "Loading..." |
| **最小显示时间** | 加载屏幕显示的最短时间（毫秒），推荐 300-800 |

> **💡 建议**：如果网站加载速度较快，可将最小显示时间设为 300ms，避免闪烁。

---

## 2. 全局背景

自定义整站的页面背景样式。

| 设置项 | 说明 |
|--------|------|
| **启用自定义背景** | 开启后可选择背景样式 |
| **背景样式** | Base 100（最浅）/ Base 200（默认）/ Base 300（较深）/ 网格背景 |

> **💡 说明**：网格背景会在页面显示微妙的网格线条，适合技术风格博客。

---

## 3. 悬浮控制栏 (Floating Dock)

页面右侧或底部的悬浮快捷工具栏。

| 设置项 | 说明 |
|--------|------|
| **启用悬浮 Dock** | 是否在通用页面显示 |
| **Dock 样式** | `底部居中` 或 `右侧可折叠` |
| **Dock 菜单** | 选择在 Dock 中显示的菜单 |
| **返回顶部按钮** | 是否显示"返回顶部"快捷按钮 |

> **💡 菜单图标设置**：在 Halo 后台的 **菜单管理** 中，编辑菜单项的 **元数据**，添加 `icon` 字段并填入 SVG 代码即可。

---

## 4. 全局侧边栏

配置侧边栏的位置和小工具。各页面可单独覆盖这些设置。

| 设置项 | 说明 |
|--------|------|
| **侧边栏位置** | `左侧` 或 `右侧` |
| **小工具列表** | 拖拽排序、添加或删除小工具 |
| **热门文章数量** | 热门文章小工具显示的数量（3-10） |
| **最新文章数量** | 最新文章小工具显示的数量（3-10） |

### 可用小工具

| 小工具 | 说明 |
|--------|------|
| **欢迎卡片（天气）** | 显示问候语和天气信息 |
| **作者卡片** | 显示博主头像、简介、社交链接 |
| **博客统计** | 显示文章数、分类数、标签数等统计 |
| **文章分类** | 显示分类列表 |
| **标签云** | 以标签云形式显示标签 |
| **热门文章** | 按浏览量排序的文章列表 |
| **最新文章** | 按发布时间排序的文章列表 |

---

## 5. 全局字体

配置整站的字体风格。

| 设置项 | 说明 |
|--------|------|
| **正文字体** | `系统默认` / `霞鹜文楷亮` / `思源宋体` |
| **字体 CDN 地址** | 自定义字体文件的 CDN 根路径 |

> **💡 说明**：霞鹜文楷适合文艺风格博客，思源宋体适合正式排版。如使用自定义 CDN，确保字体文件路径正确。
