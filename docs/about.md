# 关于页面设置 (About)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 关于**

> **💡 数据共享说明**：此处配置的「个人信息」和「GitHub 设置」将同时用于首页侧边栏的作者卡片和 GitHub 热力图

---

## 1. 个人信息

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **头像** | 留空则使用站点 Logo | - |
| **昵称** | 留空则使用站点名称 | "Sky" |
| **角色/职位** | 如：全栈开发者 | "全栈开发者 & 开源爱好者" |
| **个人简介** | 支持 HTML 标签 | - |

---

## 2. GitHub

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **GitHub 用户名** | 用于获取统计数据和贡献日历 | "sky121666" |
| **显示统计卡片** | 显示 GitHub Stats 和 Top Languages 卡片 | 开启 |
| **显示贡献热力图** | 显示 GitHub Contributions 日历图 | 开启 |
| **Stats 服务地址** | 可替换为自部署地址 | "https://github-readme-stats.xoku.cn" |
| **语言（可选）** | 设置 GitHub Stats 卡片的显示语言 | 默认（英文） |
| **主题（可选）** | 留空使用自动适配主题 | 自动适配 |

### 支持的语言
默认（英文）、简体中文、繁体中文、日语、韩语、德语、法语、西班牙语、俄语、葡萄牙语（巴西）

### 支持的主题
自动适配（推荐）、默认、透明、Dark、Radical、Merko、Gruvbox、Tokyonight、Onedark、Cobalt、Synthwave、Dracula、Nord

## 7. 显示开关

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示站点统计** | 显示文章数、分类数、评论数、访问量 | 开启 |
| **显示文章发布热力图** | 显示最近一年文章发布日期的热力图 | 开启 |
| **显示自定义内容** | 显示单页面编辑器中的内容（使用文章样式渲染） | 开启 |
| **启用评论** | 启用评论区 | 开启 |

---

## 8. 位置地图

使用 Leaflet 开源地图展示位置（自动从 CDN 加载）。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **显示位置地图** | 是否显示 | 开启 |
| **纬度** | 如：39.9042（北京天安门） | 39.9042 |
| **经度** | 如：116.4074（北京天安门） | 116.4074 |
| **位置名称** | 如：北京·中国 | "北京·中国" |
| **位置描述** | 简单描述一下这个位置 | "在这座充满活力的城市中工作和生活" |

> **💡 获取坐标**：https://lbs.amap.com/tools/picker
