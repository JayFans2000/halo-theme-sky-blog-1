# 文档设置 (Docs)

> 路径：**外观 → 主题 → Sky Blog → 设置 → 文档**

需安装 `plugin-docsme` 文档插件。

---

## 1. 文档内容页 (doc.html)

### 分享设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用分享** | 文档页面的分享按钮 | 开启 |
| **启用的分享平台** | 选择分享平台，可拖拽排序 | 微信、微博、X、QQ |

### Dock 设置

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| **启用定制 Dock 设置** | 开启后可自定义 Dock 样式和菜单 | 关闭 |
| **Dock 样式** | 底部居中 / 右侧可折叠 | 底部居中 |
| **Dock 菜单** | 选择菜单 | primary |
| **返回顶部按钮** | 是否显示 | 开启 |
| **启用目录抽屉** | 屏幕较小时可从 Dock 打开目录 | 开启 |
| **启用菜单抽屉** | 屏幕较小时可从 Dock 打开左侧文档树 | 开启 |
